[#required-package-updates]
= Required Package Updates

If the following packages are used in your project, update them to ensure compatibility with Angular 20:

[cols="2,2,3"]
|===
| Package | Required Version | Notes

| `@ngx-translate/core`
| `^17.0.0`
| Updated for Angular 20 compatibility

| `@ngx-translate/http-loader`
| `^17.0.0`
| Compatible with @ngx-translate/core v17

| `@primeng`
| `^20.3.0`
| Updated for Angular 20 compatibility

| `@primeng/themes`
| `^20.0.0`
| Updated for Angular 20 compatibility

| `ngrx-store-localstorage`
| `^20.0.0`
| Updated for Angular 20 compatibility

| `zod`
| `^4.0.0`
| Updated for Angular 20 and `@onecx/ngrx-accelerator` compatibility

| `@angular-architects/module-federation`
| `^20.0.0`
| Updated for Angular 20 compatibility

| `ngx-build-plus`
| `^20.0.0`
| Angular 20 compatible version
|===

[#installation-commands]
== Installation Commands

Run these commands to update the supporting packages:

[source, bash]
----
# Update translation packages
npm install @ngx-translate/core@^17.0.0
npm install @ngx-translate/http-loader@^17.0.0

# Update UI components
npm install @primeng@^20.3.0
npm install @primeng/themes@^20.0.0

# Update state management
npm install ngrx-store-localstorage@^20.0.0

# Update validation / schema utilities
npm install zod@^4.0.0

# Update Module Federation tooling
npm install @angular-architects/module-federation@^20.0.0

# Update build tools
npm install ngx-build-plus@^20.0.0
----

# @onecx/angular-webcomponents

`@onecx/angular-webcomponents` manages Angular webcomponents in OneCX applications, enabling component sharing.
More information about OneCX libraries can be found in the [OneCX documentation](https://onecx.github.io/docs/documentation/current/onecx-portal-ui-libs/index.html).

## Installation

```bash
npm install @onecx/angular-webcomponents
```

## Additional Commands
- `npx nx run angular-webcomponents:build` - Builds the library and outputs the result to the `dist` folder.
- `npx nx run angular-webcomponents:build-migrations` - Builds the migration files for the library.
- `npx nx run angular-webcomponents:test` - Runs the unit tests for the library.
- `npx nx run angular-webcomponents:lint` - Lints the library's codebase.
- `npx nx run angular-webcomponents:release` - Releases a new version of the library to npm, following semantic versioning guidelines.
